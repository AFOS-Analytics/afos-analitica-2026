# AFOS Analytics — Archive Branch

Snapshots diários de análises (cards + criteriosa) preservados automaticamente.
Gerado pelo workflow `.github/workflows/archive-analysis.yml`.

**Não editar manualmente** — alterações serão sobrescritas.

## Estrutura

```
archive/
└── YYYY-MM-DD/
    ├── analysis-cards.json
    └── analysis-criteriosa.json
```
